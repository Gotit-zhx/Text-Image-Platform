# 图文分享平台（Vue + Django + ALS推荐）详细文档

## 1. 项目概览
本项目由三部分组成：
- `frontend_vue`：Vue3 + TypeScript 前端（图文流、详情、发布、个人中心）
- `backend_django`：Django 后端（认证、帖子、评论、互动、推荐接口）
- `recommend_system`：ALS 推荐系统（隐式反馈矩阵分解）

当前系统已实现：
1) 前后端接口联调
2) Session 登录态（Cookie）
3) 推荐系统从数据库导出交互后每日训练
4) 推荐接口在线加载模型，失败回退热门

---

## 2. 目录结构（核心）
- `frontend_vue/src/api/*`：前端接口封装
- `frontend_vue/src/composables/*`：前端状态管理
- `backend_django/community/models.py`：业务数据模型
- `backend_django/community/views.py`：API 实现
- `backend_django/community/recommendation.py`：推荐服务加载与调用
- `backend_django/community/recsys_pipeline.py`：导出训练数据与触发训练
- `backend_django/community/management/commands/train_recsys.py`：每日训练命令
- `backend_django/community/management/commands/seed_demo_data.py`：演示数据初始化
- `recommend_system/recsys/*`：ALS 训练、评估、推理模块

## 4. 快速启动
### 4.1 后端（Django）
```bash
cd /vifile/tip/backend_django
conda activate tip
pip install -r requirements.txt
/home/zhx/miniconda3/envs/tip/bin/python manage.py migrate
/home/zhx/miniconda3/envs/tip/bin/python manage.py seed_demo_data
/home/zhx/miniconda3/envs/tip/bin/python manage.py runserver 127.0.0.1:8000
```

### 4.2 前端（Vue）
在前端 `.env` 中配置：
```env
VITE_API_BASE_URL=http://127.0.0.1:8000/api
VITE_USE_MOCK_API=false
```
启动：
```bash
cd /vifile/tip/frontend_vue
npm install
npm run dev
```

## 5. 推荐系统训练（数据库导出）
### 5.1 一次训练
```bash
cd /vifile/tip/backend_django
/home/zhx/miniconda3/envs/tip/bin/python manage.py train_recsys --trials 10 --val-ratio 0.1 --min-rows 10
```

## 7. 已实现的关键安全与稳定策略
1. 未登录访问受限 API 返回 401 JSON（非 302 页面跳转）
2. 前端请求默认 `credentials: include`，适配 Django Session
3. 前端请求层支持超时与统一错误提示
4. 推荐接口模型不可用时回退热门内容
5. 推荐训练支持最小样本阈值，样本不足自动跳过训练并保留旧模型
6. 内容渲染已接入 `DOMPurify` 净化策略

---

## 8. 联调验收清单（最小）
1. 登录/注册成功，刷新后登录态保留
2. 未登录互动触发登录弹窗
3. 非作者编辑帖子被拦截（前后端双重）
4. 搜索支持服务端优先，失败本地回退
5. 推荐接口可返回列表（无模型时回退热门）
6. 训练命令执行后状态文件更新为 `success`

## 9. 常见问题
### 9.1 `ModuleNotFoundError: django`
未在 `tip` 环境执行或依赖未安装。

### 9.2 训练报样本太少
执行：
```bash
python manage.py seed_demo_data
python manage.py train_recsys --min-rows 1
```

### 9.3 推荐接口无个性化结果
可能模型尚未训练或用户交互不足。先检查：
- `recsys_train_status.json`
- `recommend_system/cache/trained_model.npz`

### 9.4 前端请求 401/403
检查：
- 前端是否使用 `VITE_USE_MOCK_API=false`
- 是否携带 Cookie（已在 client 默认开启）
- 后端服务是否同源/允许跨域
