# recommend_system 功能说明（论文模块描述）

## 1. 模块目标
基于隐式反馈构建个性化推荐模型，输出按用户排序的帖子候选。

## 2. 方法概述
1. 从行为日志构建用户-物品交互矩阵。
2. 使用 ALS 进行矩阵分解学习。
3. 使用 Optuna 搜索部分超参数。
4. 输出 Top-K 推荐结果供后端接口使用。

## 3. 输入与输出
- 输入：`daily_interactions.csv`（含 `read/like/favorite/share`）。
- 输出：`trained_model.npz` + 映射缓存文件。

## 4. 核心代码
- `recsys/data.py`：读取、过滤、ID 映射。
- `recsys/trainer.py`：训练、验证、推理。
- `recommender.py`：训练命令入口。

## 5. 与业务系统集成
后端通过 `recsys_pipeline.py` 调用训练，通过 `recommendation.py` 加载模型并服务在线推荐。

## 6. 稳定性策略
1. 小样本阈值保护。
2. 切分失败时随机切分回退。
3. 在线推荐异常时热门回退。
