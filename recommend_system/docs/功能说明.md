# recommend_system 功能说明

## 1. 功能定位
- 提供离线训练与在线推理能力：
  - 数据清洗
  - ALS 训练
  - 超参搜索（Optuna）
  - 推荐结果生成

## 2. 核心模块
- `recsys/config.py`：训练配置与阈值
- `recsys/data.py`：数据加载与过滤
- `recsys/trainer.py`：训练/验证/模型输出
- `recsys/predict.py`：推理与候选打分
- `recommender.py`：训练入口与服务桥接

## 3. 输入输出
- 输入：`daily_interactions.csv`（来自业务库导出）
- 输出：`trained_model.npz`（ALS 模型参数）

## 4. 与后端集成关系
- Django 通过 `recsys_pipeline.py` 触发训练
- 在线推荐由 `community/recommendation.py` 读取模型并对外提供接口

## 5. 失败回退
- 模型不可用或推理异常时，后端回退热门帖推荐
