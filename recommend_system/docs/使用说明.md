# recommend_system 使用说明（复现实验版）

## 1. 数据输入
训练输入由后端导出，标准列为：
`user_id, article_id, read, like, favorite, share`

## 2. 推荐训练（推荐方式）
```bash
cd backend_django
python3 manage.py train_recsys --trials 10 --val-ratio 0.1 --min-rows 10
```

## 3. 参数说明
- `trials`：Optuna 搜索轮数。
- `val-ratio`：验证集比例。
- `min-rows`：训练最小样本阈值。

## 4. 产物检查
- `recommend_system/cache/daily_interactions.csv`
- `recommend_system/cache/trained_model.npz`
- `backend_django/recsys_train_status.json`

## 5. 故障排查
1. 样本不足：先增加用户行为或降低 `min-rows`。
2. 训练异常：检查 CSV 列是否完整。
3. 推荐异常：检查模型文件是否存在并已更新。
