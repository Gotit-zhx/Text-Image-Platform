# backend_django 功能说明

## 1. 功能概览
- 认证：注册、登录、退出、当前用户
- 社区：初始化、搜索、推荐
- 内容：发帖、编辑、评论
- 互动：点赞、收藏、关注、评论点赞、删评
- 推荐：数据库导出交互 -> ALS 训练 -> 在线推荐

## 2. 核心模块
- `community/models.py`：业务实体模型
- `community/views.py`：HTTP API 实现
- `community/http.py`：统一响应
- `community/recommendation.py`：推荐模型加载与回退
- `community/recsys_pipeline.py`：导出训练数据与触发训练
- `community/management/commands/train_recsys.py`：训练命令
- `community/management/commands/seed_demo_data.py`：演示数据初始化

## 3. 推荐链路
1. 从数据库导出 `daily_interactions.csv`
2. 调用 `recommend_system/recommender.py` 训练
3. 生成 `trained_model.npz`
4. 推荐接口按用户输出结果，失败回退热门

## 4. 统一响应
返回结构：
```json
{"code":0,"message":"ok","data":{},"requestId":"..."}
```

## 5. 运维可观测
- 训练状态文件：`recsys_train_status.json`
- 训练日志可写入 `recsys_train.log`
