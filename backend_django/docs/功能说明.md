# backend_django 功能说明（论文模块描述）

## 1. 模块定位
后端负责业务规则、权限控制、数据持久化、推荐训练触发与推荐结果服务。

## 2. 功能清单

### 2.1 认证与会话
- 用户注册、登录、退出、会话恢复。
- 管理员登录与社区用户会话隔离。

### 2.2 社区业务
- 首页初始化（帖子、评论、互动状态、通知）。
- 帖子发布、编辑、评论、删除评论。
- 互动行为：`read/like/favorite/share/follow/comment-like`。

### 2.3 后台管理
- 帖子审核（单条与批量）。
- 评论隐藏/恢复/删除。
- 用户角色管理。
- 审计日志查询。

### 2.4 推荐能力
- 导出行为数据为 `daily_interactions.csv`。
- 触发 ALS 离线训练并写入状态文件。
- 在线推荐接口按用户返回结果，异常回退热门。

## 3. 核心代码位置
- `community/models.py`：实体模型与字段。
- `community/views.py`：API 与业务流程。
- `community/recsys_pipeline.py`：导出与训练任务。
- `community/recommendation.py`：在线推荐加载与回退。
- `community/management/commands/train_recsys.py`：训练命令入口。

## 4. 响应规范
统一响应结构：
```json
{"code":0,"message":"ok","data":{},"requestId":"..."}
```

## 5. 可观测性
- 训练状态：`recsys_train_status.json`
- 训练日志：`recsys_train.log`
- 审计日志：`AuditLog` 表
