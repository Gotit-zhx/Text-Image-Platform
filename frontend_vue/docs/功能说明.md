# frontend_vue 功能说明

## 1. 功能概览
- 首页信息流（推荐/热门/更新/关注）
- 搜索（服务端优先，失败回退本地）
- 图文详情与评论互动
- 发布与编辑图文
- 个人中心（帖子/评论/收藏/粉丝/关注/资料编辑）
- 登录/注册/退出

## 2. 关键模块
- `src/App.vue`：页面编排与路由态消费
- `src/composables/useAuthState.ts`：鉴权状态与会话动作
- `src/composables/useCommunityState.ts`：帖子/评论/互动状态
- `src/api/client.ts`：请求封装（超时、错误处理）
- `src/api/auth.ts`、`src/api/community.ts`：业务 API

## 3. 路由与状态
- 路由驱动页面：home/profile/publish/edit/detail
- URL Query 负责恢复状态：`nav`、`q`、`menu`、`focusTs`

## 4. 安全策略
- 详情页 `v-html` 已净化（DOMPurify）
- 未登录互动统一拦截并引导登录
- 编辑操作增加作者权限校验

## 5. 推荐与搜索
- 推荐：调用 `/community/recommend`
- 搜索：调用 `/community/search`
- 两者均提供失败回退策略
