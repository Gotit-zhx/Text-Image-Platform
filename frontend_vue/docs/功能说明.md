# frontend_vue 功能说明（论文模块描述）

## 1. 功能概览
1. 社区首页：`推荐/热门/更新/关注` 四导航。
2. 内容详情：图文阅读、评论、点赞、收藏、分享。
3. 创作链路：发布与编辑帖子。
4. 个人中心：帖子、评论、收藏、粉丝、关注、通知、资料编辑。
5. 后台入口：管理员登录与内容治理。

## 2. 关键实现
- `src/App.vue`：路由态与页面编排、分享逻辑。
- `src/composables/useAuthState.ts`：会话恢复与登录流程。
- `src/composables/useCommunityState.ts`：社区主状态与行为上报。
- `src/api/community.ts`：社区、推荐、行为采集接口。

## 3. 行为采集
前端已接入行为采集：
- 阅读：进入详情页上报 `read`。
- 点赞：点击点赞上报 `like`。
- 收藏：点击收藏上报 `favorite`。
- 分享：原生分享或复制链接后上报 `share`。

## 4. 交互特点
1. 推荐接口失败时自动回退热门。
2. 未登录互动统一拦截并拉起登录弹窗。
3. 分享支持 `Web Share API`，并兼容剪贴板降级。
