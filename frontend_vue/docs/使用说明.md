# frontend_vue 使用说明（复现实验版）

## 1. 环境要求
- Node.js 18+
- npm 9+

## 2. 环境变量
在 `frontend_vue/.env` 中配置：
```env
VITE_API_BASE_URL=http://127.0.0.1:8000/api
VITE_USE_MOCK_API=false
```

## 3. 启动命令
```bash
cd frontend_vue
npm install
npm run dev
```

## 4. 推荐验证路径
1. 登录用户进入首页“推荐”。
2. 浏览帖子详情触发 `read` 行为。
3. 执行点赞/收藏/分享。
4. 后端训练后再次打开推荐页，观察排序变化。

## 5. 构建与检查
```bash
cd frontend_vue
npm run build
```

## 6. 常见问题
1. 401/403：检查后端是否运行、Cookie 是否携带。
2. 分享失败：浏览器不支持原生分享时会自动复制链接。
3. 推荐无变化：确认后端已训练并加载新模型。
