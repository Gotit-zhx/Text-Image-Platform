# frontend_vue 使用说明

## 1. 依赖与环境
- Node.js 18+
- npm 9+

## 2. 环境变量
在前端目录创建 `.env`：
```env
VITE_API_BASE_URL=http://127.0.0.1:8000/api
VITE_USE_MOCK_API=false
```

## 3. 启动
```bash
cd /vifile/tip/frontend_vue
npm install
npm run dev
```

## 4. 常用验证路径
1. 注册/登录 -> 刷新页面验证会话恢复
2. 首页推荐与搜索 -> 查看 URL 参数变化与回退
3. 进入详情 -> 评论/点赞/收藏/关注
4. 个人中心 -> 编辑资料、编辑帖子

## 5. 常见问题
- 401：确认后端在运行，且 `VITE_USE_MOCK_API=false`
- 跨域问题：确认后端开发 CORS 中间件已启用
- 推荐为空：先在后端执行推荐训练命令

## 6. 自动化测试与排查
执行前端自动化检查：
```bash
cd /vifile/tip/frontend_vue
npm run test
```

说明：
- `npm run test` 当前会执行 `type-check + build-only`，用于快速发现类型与构建问题。

若失败，优先排查：
1. 依赖是否安装完整（`npm install`）
2. Node 版本是否满足要求
3. 是否有 API 类型定义与组件类型冲突
